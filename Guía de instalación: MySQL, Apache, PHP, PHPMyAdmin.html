Guía de instalación: MySQL, Apache, PHP, PHPMyAdmin

by: asuna-yuukiuwu


!!! Es importante llevar este orden para evitar futuros errores de configuración al momento de ejecutarlo.


-----------------------------------------------------------------------------------------------


* Pasos para instalar el servidor "Apache":


Abrimos nuestra terminal y ejecutamos:

$ sudo apt update

$ sudo apt upgrade

Esto se hace con el fin de actualizar los paquetes y que no cause problemas a la hora de instalar los programas.


- Ya que tenemos todos los paquetes actualizados, ejecutamos:

$ sudo apt install apache2

(contestan con "s" a continuar y aprietan la tecla enter)

- Para verificar que tenemos instalado el servidor, nos dirigimos a nuestro navegador y en la barra de direcciones escriben:

http://localhost/

Al buscarlo, debería de arrojar la información acerca de "Apache2" con el mensaje: "It works!".


!!! Para solucionar problemas de permisos con la carpeta "html":


Esta carpeta es la que se crea automáticamente al instalar el servidor Apache, y es donde alojaremos nuestros proyectos.

Pero, esta viene con los permisos restringidos, por lo que no podemos ni crear, editar o eliminar archivos. Es por ello, que debemos de solucionar estos problemas de permisos con los siguientes comandos:


- Abrimos terminal en la ubicación de la carpeta "www" llendo a esa carpeta, presionando click derecho y en “abrir en una terminal”.

- Estando en la terminal, debe de tener la siguiente ubicación:

nombre@nombre:~/var/www$

- Escribimos los siguientes comandos:

$ sudo chmod 777 html/

Con este comando resolvemos el problema de permisos en la carpeta "html", para que puedamos crear, editar y eliminar libremente archivos. :)


-----------------------------------------------------------------------------------------------


* Pasos para instalar el servidor de base de datos "MySQL":


- Abrimos nuestra terminal y ejecutamos:

$ sudo apt install mysql-server

(contestan con "s" a continuar y aprietan la tecla enter)

Por el momento, solo ejecutamos este comando y continuamos con "PHP".


-----------------------------------------------------------------------------------------------


* Pasos para instalar "PHP":


- En nuestra terminal ejecutamos:

$ sudo apt install php

(contestan con "s" a continuar y aprietan la tecla enter)


Por el momento, solo ejecutamos este comando y continuamos con "PHPMyAdmin".


-----------------------------------------------------------------------------------------------


* Pasos para instalar "PHPMyAdmin":


- En nuestra terminal ejecutamos:

$ sudo apt install phpmyadmin

(contestan con "s" a continuar y aprietan la tecla enter)

Les va a arrojar la opción del servidor a usar. Para ello seleccionarán "apache2" apretando la tecla de espacio. Este debería de marcar un * en la opción:

[*] apache2

[ ] lighttpd

Y le dan "aceptar".


- Posterior a ello (después de que cargue un rato) les aparecerá que se debe de tener una base de datos ya instalada y configurada. Solo den en "aceptar".

Preguntará si desean configurar la base de datos. Solo den en "Si".


- Les pedirá que configuren una contraseña para "phpmyadmin". Mi recomendación es que pongan algo corto para que no se les olvide.

Una vez que terminan de confirmar la contraseña, seguirá el proceso hasta que termine la instalación.


-----------------------------------------------------------------------------------------------


!!! Obteniendo permisos poder crear bases de datos:


- Abrimos nuestra terminal y ejecutamos:

$ sudo -i

De esa manera entramos al usuario "root". Debe de aparecer de la siguiente manera: root@nombre:~#


- Una vez en root, ejecutamos:

mysql -u root -p

Solicitará la contraseña que hayan creado anteriormente en "phpmyadmin".


- Una vez entren a "mysql>", ejecutan los siguientes comandos:

create user 'nombredelusuario'@'%' identified by 'contraseña'

(aprietan enter)


Explicación: 'nombredelusuario' -> cambiarlo al nombre de usuario que desean tener.

'contraseña' -> poner la contraseña que se creó para phpmyadmin


- Ahora, se le dará al usuario que acaban de crear todos los privilegios para que pueda crear bases de datos:

grant all privileges on *.* to 'nombredelusuario'@'%' with grant option;

(aprietan enter)

- Y para finalizar:

flush privileges

(aprietan enter)


Para verificar que todo esté correcto, ingresen al navegador y en la barra de direcciones escriben:

http://localhost/phpmyadmin

(escriben el usuario que crearon y la contraseña)

Posteriormente se dirigen al apartado de "Bases de datos". Si todo salió bien, no debería de arrojar ni un error al ingresar al apartado. :)


Ahora, iniciamos con la instalación de "MySQL Workbench".


-----------------------------------------------------------------------------------------------


* Pasos para instalar "MySQL Workbench":


- Abrimos la terminal y ejecutamos:

sudo snap install mysql-workbench-community

(contestan con "s" a continuar y aprietan la tecla enter)

Con esto, ya deberíamos tener en nuestras aplicaciones "MySQL Workbench".

Al abrirlo, ya debería estar creada una conexión llamada "Local instance 3306", pero tendremos un problema al querer abrirlo, ya que el usuario por defecto es "root". Haremos lo siguiente:


- Para ello, ubicamos donde dice "MySQL Connection" y a un lado de ello, habrán dos botones: un signo de "+" y otro de un herramienta.


- Seleccionamos el de herramienta y se nos abrirá una pestaña en donde solo cambiaremos el "username" al que creamos para phpmyadmin.


- En "password" solo apretamos en "clear" y cerramos la ventana en "close".


Con esto, ya deberíamos poder ingresar a la instancia local ya creada con la contraseña que creamos en "phpmyadmin".




¡Y listo! Ya tenemos configurado nuestro MySQL Workbench, Apache y PHPMyAdmin. :)
